Política de Privacidade

LT Localizações valoriza sua privacidade e está comprometida em proteger as informações pessoais coletadas através de nosso aplicativo. Esta política detalha como coletamos, usamos e protegemos os dados dos usuários.

1. Informações Coletadas

1.1 Dados de Login

Nosso aplicativo utiliza o Firebase Authentication para autenticação de usuários. Durante o processo de login, coletamos seu e-mail e senha ou dados de login fornecidos por terceiros, como Google, conforme suportado pelo Firebase.

1.2 Consulta de Placas

As consultas realizadas para buscar informações sobre placas de veículos são enviadas ao Firebase Realtime Database. Não armazenamos registros das consultas individuais realizadas pelo usuário.

2. Uso das Informações

2.1 Autenticação

As informações de login são usadas exclusivamente para autenticar e autorizar o acesso ao aplicativo.

2.2 Consultas

Os dados enviados para o banco de dados servem apenas para fornecer informações relacionadas à placa pesquisada, sem armazenamento de histórico de consultas.

3. Compartilhamento de Informações

Não compartilhamos informações pessoais com terceiros. Os dados armazenados e processados pelo Firebase são geridos conforme a Política de Privacidade do Firebase.

4. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados dos usuários. No entanto, a segurança absoluta não pode ser garantida.

5. Direitos do Usuário

5.1 Acesso e Correção

Você pode solicitar o acesso, correção ou exclusão de suas informações entrando em contato conosco.

5.2 Cancelamento

A desinstalação do aplicativo interrompe o uso dos dados pessoais.

6. Alterações

Reservamo-nos o direito de modificar esta política a qualquer momento. Recomendamos revisá-la periodicamente.
